# UART Communication using Embedded C

## Description
This project demonstrates serial communication using the UART protocol
to transmit and receive data between a microcontroller and a PC.

## Microcontroller / Board
Arduino UNO / ESP32 / STM32

## Programming Language
Embedded C

## Tools Used
Arduino IDE / Serial Monitor

## Key Features
- UART initialization and configuration
- Data transmission and reception
- Baud rate configuration

## What I Learned
- Working principles of UART protocol
- Serial communication basics
- Debugging using serial monitor
